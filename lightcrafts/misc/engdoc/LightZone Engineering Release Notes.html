LightZone Engineering Release Notes

Note: This is text file for engineers only. Please feel free to be verbose.

Changes for 3.1, rXXXX, 3-Sep-2007

New Features

  • You can now restrict every tools' effect to a range of color or luminosity.
  • The Raw Adjustments tool now automatically removes "hot pixels" from your raw photos.
  • Crop and Rotate have been improved.

Camera Support

  • Raw files from the following cameras are now supported: Olympus E-400, E-410, E-510; Ricoh Caplio GX100.

Changes for 3.0.6, rXXXX, 28-June-2007

Fixes (all platforms)

  • Upgrading from pre-3.0 versions of LightZone Basic to 3.0.x LightZone Basic now works properly.
  • A DNG image created from a Fuji raw image was not handled correctly. This has been fixed.

Fixes (Windows only)

  • The font used in menus was too big. This has been fixed.

Changes for 3.0.5, r8780, 27-June-2007

New Features

  • LightZone Basic is now available again.

Fixes

  • Some Sony raw photo were opened with the wrong dimensions. This has been fixed.
  • In some cases, you would get a Save dialog even though the Auto Save preference was enabled.

Camera Support

  • Kodak P712, Nikon D40X, Olympus SP-550UZ, and Samsing GX10 raw files are now supported.

Other changes

  • If a photo is read-only, the ability to edit metadata is now disabled.

Changes for 3.0.2, r8648, 8-June-2007

Fixes

  • Fixed oversaturated colors in the sharpening tool.
  • Fixed related issue in noise reduction.

Changes for 3.0.1, r8637, 7-June-2007

New Features

  • Adjusting the Exposure slider of the Raw Adjustments tool so that the value is negative performs highlight recovery.

Fixes

  • Removed artifacts that were sometimes introduced when using the Relight tool.
  • The Black & White and Color Balance tools have had minor corrections.
  • In some cases, vibrant colors were overly saturated. This has been fixed.
  • Editing of metadata works much better.
  • Fixed generation of IPTC and XMP metadata for JPEG files that previously resulted in an incompatibility with Photoshop CS3.
  • Fixed obtaining the artist information from CR2 files.
  • A photo's rating is now properly stored when generating new TIFF files.
  • Fixed obtaining the exposure-time/shutter-speed information from Canon raw files.
  • Fixed reading of artist, copyright, caption, and rating metadata from DNG files.
  • Styles now appear in the Import Styles dialog.
  • The capture date/time from CRW files is now included when converting to JPEG.
  • Now writing IPTC metadata in a more standards-conformant way using the Dublin Core XML schema.

Camera Support

  • Fuji Finepix S5 Pro raw files are now supported.

Other changes

  • LightZone now shows both the "Edit Size" and the "Version Size" for an edited photo and its original.
  • LightZone now goes to the "Discover LightZone" help page instead of the cover when first launched.
  • You can now sort by file type in the browser.
  • The Browser menu now includes Select Recent and Unselect All menu items.
  • You can now use the Delete/Backspace key to delete a region while editing a region.
  • When converting to JPEG, lens metadata is now preserved.

Changes for 3.0, r8440, 21-May-2007

User Interface

  • LightZone now has distinct browse and edit modes.
  • There is now context-sensitive help throughout LightZone.

Photo Browser

  • Related photos can now be stacked.
  • You can now move photos via drag & drop to folders in the Folder Navigator.
  • You can now select only the most recent versions of your photos in a given folder.
  • You can now edit IPTC metadata.

Photo Editor

  • LightZone now comes bundled with many ready-made Styles.
  • The Relight tool (an improved ToneMapper tool) has new Depth and Fuzz sliders.
  • There is now a History tab that shows you all the edits you have make to your photo since opening it for the current editing session.
  • The Raw Adjustments tool has a new As Shot button.
  • LightZone no longer uses LZN files and now always saves your work as either JPEG or TIFF files for maximum interoperability with other applications.

Camera Support

  • Canon EOS 1D Mark III raw files are now supported.

Changes for 2.4, rXXXX, XX-Apr-2007

New features

Changes for 2.2, r7421, 5-Mar-2007

New requirements (Mac OS X only)

  • Java 1.5 Release 5 is now required. If you have not already updated to it, get it by selecting Software Update from the Apple menu.

New features

  • You can now rate your photos.
  • Raw conversion has been improved.

  • Orientation changes (and ratings) are now stored in sidecar XMP files for interoperability with other applications.

    In previous versions of LightZone, orientation information was stored in a LightZone database. To preserve the orientation of photos you've already rotated, the first time your browse a folder with LightZone 2.2, it will create a new sidecar XMP file for every photo in that folder that's been rotated using LightZone's browser. If the folder is read-only, no XMP files can be created so orientation information will be lost for photos in that folder.

  • Printer settings are now saved with your work so you can easily reproduce prints.
  • The user interface has a darker and all-gray appearance to reduce color distractions.

Camera support

  • Nikon D40 raw files are now supported.

Fixes

  • All known printing problems have been fixed.
  • Fixed a problem where rotating a photo would also rotate all saved versions derived from that photo.
  • Fixed a compatibility problem between sidecar TIFF files created by LightZone and opening them in other applications.
  • Fixed a crash with some display color profiles.
  • Fixed artifacts with the noise reduction and Tone Mapper tools.

Other changes

  • Performance on computers having Intel (or compatible) processors has been improved.
  • The way preview images are generated has changed. The first time you click on a thumbnail, it may take several seconds for the preview to appear.
  • XMP metadata embedded inside JPEG files is now read.
  • The browser menu previously available only via right/control click has been given its own menu in the menu bar.
  • You can now edit a photo simply by clicking on its preview image.
  • The behavior of the "eye" button that shows the before and after versions of your photo has changed: you now need to click and hold the button to see the before version.
  • Support for PNG images has been removed.
