LightZone Release Notes

Note: The complete documentation for LightZone is integrated into the application. To access it, from within LightZone, select Help > LightZone Help. Mac OS X users can alternatively click here.

Changes after 4.0.0

Detailed change log is available on github release page.

Changes for 4.0.0

  • LightZone is open-source software released under BSD license
  • Copy protection removed (Pavel Benak)
  • A lot of fixes for building on all recent platforms (Windows, Linux, Mac OS) (Masahiro Kitagawa, Pavel Benak, Visa Jokelainen, Doug Pardee)
  • Libraries upgrade - libjpeg replaced with libjpeg-turbo for faster JPEG loading and saving (Masahiro Kitagawa)
  • dcraw updated to latest version (Doug Pardee)
  • On Linux platform icons and desktop files were added (Andreas Rother)
  • On Linux platform LightZone memory is allocated dynamically based on amount of RAM (Andreas Rother)
  • On Linux platform LightZone is distributed as packages for distributions (Masahiro Kitagawa, Andreas Rother)
  • Fixed luminance selection problem on Linux platform (Michael Schulze)
  • Fixed program crash on Linux platform when loading RAW files (Masahiro Kitagawa)
  • Fixed problem with saving files into directories with more than one byte characters in UTF-8 encoding (Pavel Benak, Doug Pardee)
  • Fixed problem with saving files into directories with four byte UTF-8 characters (Masahiro Kitagawa)
  • Fix for line artifacts on Linux platform (Masahiro Kitagawa)
  • Fix for size of exported files (Masahiro Kitagawa)
  • Translations added:
    • Japanese (Masahiro Kitagawa)
    • Spanish (Roberto Quintero)
  • Added RAW tone curves for cameras from LightZombie project (Doug Pardee)

Changes for 3.0.5

New Features

  • LightZone Basic is now available again.

Fixes

  • Some Sony raw photo were opened with the wrong dimensions. This has been fixed.
  • In some cases, you would get a Save dialog even though the Auto Save preference was enabled.

Camera Support

  • Kodak P712, Nikon D40X, Olympus SP-550UZ, and Samsing GX10 raw files are now supported.

Other Changes

  • If a photo is read-only, the ability to edit metadata is now disabled.

Changes for 3.0.2

Fixes

  • Fixed oversaturated colors in the sharpening tool.

Changes for 3.0.1

New Features

  • Adjusting the Exposure slider of the Raw Adjustments tool so that the value is negative performs highlight recovery.

Fixes

  • Improved functioning of Relight tool.
  • The Black & White and Color Balance tools have had minor corrections.
  • In some cases, vibrant colors were overly saturated. This has been fixed.
  • Editing of metadata works much better.
  • Fixed generation of IPTC and XMP metadata for JPEG files that previously resulted in an incompatibility with Photoshop CS3.
  • Fixed obtaining the artist information from CR2 files.
  • A photo's rating is now properly stored when generating new TIFF files.
  • Fixed obtaining the exposure-time/shutter-speed information from Canon raw files.
  • Fixed reading of artist, copyright, caption, and rating metadata from DNG files.
  • Styles now appear in the Import Styles dialog.
  • The capture date/time from CRW files is now included when converting to JPEG.
  • Now writing IPTC metadata in a more standards-conformant way using the Dublin Core XML schema.

Camera Support

  • Fuji Finepix S5 Pro raw files are now supported.

Other Changes

  • LightZone has always edited the full size image. LightZone now displays the dimensions of the full edit size and the version size separately.
  • LightZone now goes to the "Discover LightZone" help page instead of the cover when first launched.
  • You can now sort by file type in the browser.
  • The Browser menu now includes Select Recent and Unselect All menu items.
  • You can now use the Delete/Backspace key to delete a region while editing a region.
  • When converting to JPEG, lens metadata is now preserved.

Changes for 3.0

Intelligent Tools

  • The new Relight tool replaces the already "magical" ToneMapper: new controls allow for greater precision in relighting virtually any scene.
  • The new History pane, gives users full and direct access to intermediate editing steps, all the way from first opening the photograph.
  • Added support for RAW images from the latest dSLR cameras such as the Canon 1D Mark III.

Improved Learnability

  • Styles automatically produce tools and guide new users through the interface process.
  • New context-sensitive help and tool tips to assist users pre-flight and in-flight.
  • Revamped user interface with distinct Browse and Edit modes, new skins to optimize clarity of the photograph, logical button groupings.

Improved Interoperability

  • New default save mode: edit stack now embedded in JPEG. Provides seamless interoperability with any best-of-breed application.
  • New Send and Convert capability provides advanced pre-processing capabilities to other applications in the photographic ecosystem.
  • New drag and drop capabilities in Browse mode for folders, file-system or directly to other applications in the dock (on Mac).

New Integrated Batch Processing

  • New ift Style in Browse mode and apply to multiple images at once.
  • New Send and Convert capabilities convert your images for sending by e-mail, posting on a web gallery or simply convert in another file format.
  • New drag & drop capabilities in Browse mode to folders, file-system or directly to other applications in the dock (on Mac).

New Asset Management Capabilities

  • New Stack and Unstack hides and shows the intermediate edit versions of every photograph.
  • New Smart Select button automatically selects the latest version of the photograph, so they can be post processed in batch within a single click
  • New Metadata Editor allows for editing of Title, Copyright, Creator and other fields, a new preference for Copyright will automatically populate when specified.
